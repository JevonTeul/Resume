# Jevon Teul — Resume

Personal resume web app built with Node.js, Express, EJS, and served via an MVC structure.

## Live Site

https://resume-et6h.onrender.com

## Tech Stack

- Node.js + Express
- EJS templating
- MVC architecture
- Nodemon (dev)
- Hosted on Render

## Run Locally

git clone https://github.com/JevonTeul/Resume.git
cd Resume
npm install
npm run dev
